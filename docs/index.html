Spark MCP

Read-only MCP server for Spark Desktop email. Queries Spark's local SQLite databases directly -- no IMAP connections, no API keys, no network access required.

Read-Only Access

All database access uses SQLite read-only mode. Your email can never be modified or deleted through this server.

No Credentials Needed

Reads directly from Spark's local cache on disk. No passwords, no API keys, no OAuth tokens.

Full-Text Search

Uses Spark's built-in FTS5 index to search message bodies, subjects, senders, and recipients across all accounts.

macOS Only

Works with Spark Desktop for macOS. Requires the app to be installed with at least one configured account.

Quick Start

# Clone and install
git clone https://github.com/bbdaniels/spark-mcp.git
cd spark-mcp && pip install -e .

# Register the MCP server
claude mcp add spark-email -- spark-mcp
Requirements: macOS with Spark Desktop installed, Python 3.10+, and the mcp Python package.

Tools

Tool Description
spark_list_accounts List configured email accounts
spark_search Full-text search across all accounts (supports FTS5 syntax)
spark_get_message Read a specific message with body text and attachments
spark_get_conversation Read an entire email thread in order
spark_recent List recent inbox messages, optionally filtered by account
spark_search_contacts Search contacts by name or email address
spark_list_folders List folders and labels per account

Security

  • All database access is read-only (SQLite ?mode=ro)
  • No network connections -- reads only from local disk
  • No credentials needed -- uses Spark's existing local cache
